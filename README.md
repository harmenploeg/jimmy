# Amsterdam Locaties

Deze site is statisch. Voor GitHub Pages kun je alles in de root van je repository zetten:

- `index.html`
- `app.js`
- `styles.css`

De map `assets` is niet nodig als je deze root-bestanden gebruikt.

## GitHub Pages aanzetten

1. Open je repository op GitHub.
2. Ga naar `Settings` -> `Pages`.
3. Kies `Deploy from a branch`.
4. Kies branch `main` en folder `/root`.
5. Open de Pages-link op je mobiel.

Actuele locatie werkt op mobiel betrouwbaar via HTTPS. GitHub Pages gebruikt HTTPS, dus daar kan je telefoon om locatietoestemming vragen.
